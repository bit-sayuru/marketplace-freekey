## 🚀 Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/bit-sayuru/marketplace-freekey.git
git push -u origin main
npm run deploy
```

## 💻 Development

```bash
npm install
npm run dev
```
